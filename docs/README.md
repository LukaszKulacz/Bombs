## Gra typu Bomberman
Gra typu Bomberman utworzona z pomocą silnika gier GoDot.
W grze możliwe jest użycie gamepada (dotychczas przetestowano pad od Xbox One).

Obecna wersja gry zawiera jedynie bardzo podstawowe mechaniki:
- chodzenie 
- ustawianie bomby
- niszczenie ścian
